About 02_Generate_Trajectories_for_Model_Testing_LLM_DIRECT 

This code is used to generate the trajectory data frames to be used for model testing of the LLM_DIRECT policy.
The trajectory data frames are generated using the PPO model trained with LLM_DIRECT policy for five distinct seed values.

About 03_Model_Testing_LLM_DIRECT

This code is used to test the trajectories data frames generated by using the PPO model trained with LLM_DIRECT policy. 
It calculates the misalignment, collision rate and the final reward for each trajectory data frame.
