About 01_Generate_Trajectories_for_Model_Testing_LLM_DIRECT_Edge_Case_2

This code is used to generate the trajectory data frames to be used for model testing of the LLM_DIRECT policy.
The trajectory data frames are generated using the PPO model trained with LLM_DIRECT policy for five distinct seed values in the edge case scenario 2.

About 02_Model_Testing_LLM_DIRECT_Edge_Case_2

This code is used to test the trajectories data frames generated by using the PPO model trained with LLM_DIRECT policy. 
It calculates the final misalignment, average terminate time and the average episodic reward for all the data frames.
